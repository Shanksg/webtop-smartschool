QUICK START (5 MINUTES)
================================================================================

1. Create project directory:
   mkdir smartschool-monitor && cd smartschool-monitor

2. Copy all these files into that directory

3. Create config/config.yaml:
   mkdir config
   cat > config/config.yaml << 'YAML'
   students:
     - name: "Your Child"
       username: "smartschool_username"
       password: "smartschool_password"
   YAML

4. Edit docker-compose.yml:
   - Set SCHEDULES (when to check)
   - Set NOTIFIERS (where to send notifications)
   Example: NOTIFIERS: "hassio://user@192.168.1.100/YOUR_TOKEN"

5. Start the service:
   docker-compose up -d

6. Check logs:
   docker-compose logs -f smartschool-monitor

Done! You'll get notifications at your configured times.
